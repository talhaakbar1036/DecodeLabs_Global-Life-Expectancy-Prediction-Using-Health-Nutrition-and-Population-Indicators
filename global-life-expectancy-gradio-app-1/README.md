# Global Life Expectancy Prediction Project

## Overview
The Global Life Expectancy Prediction project aims to provide insights into life expectancy based on various health, nutrition, and population indicators. This application utilizes machine learning models to predict health expenditure and forecast future trends, specifically focusing on Pakistan.

## Project Structure
```
global-life-expectancy-gradio-app
├── app.py                  # Main application file for the Gradio interface
├── assets
│   ├── styles.css          # Custom CSS for dark theme and styling
│   └── animations.js       # JavaScript for enhancing user experience with animations
├── data
│   ├── dataset
│   │   ├── de46020d-9719-41d6-a517-0a7fb19791b1_Data.csv  # Primary dataset for predictions
│   │   └── de46020d-9719-41d6-a517-0a7fb19791b1_Data (1).csv  # Supplementary dataset
│   ├── ml_prediction
│   │   ├── health_expenditure_classification_predictions.csv  # Predictions for health expenditure classification
│   │   ├── infant_mortality_predictions.csv  # Predictions for infant mortality rates
│   │   └── pakistan_health_expenditure_forecasts.csv  # Forecasts for health expenditure in Pakistan
│   └── saved_ml_models
│       ├── lstm_pakistan_health_expenditure.h5  # Saved LSTM model for predictions
│       └── prophet_pakistan_health_expenditure.json  # Saved Prophet model for forecasting
├── requirements.txt        # Dependencies required to run the application
└── README.md               # Documentation for the project
```

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   cd global-life-expectancy-gradio-app
   ```

2. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

3. Run the application:
   ```
   python app.py
   ```

4. Open your web browser and navigate to `http://localhost:7860` to access the Gradio interface.

## Usage Guidelines
- Input your health and nutrition indicators into the provided fields.
- Click on the "Predict" button to receive predictions regarding health expenditure and life expectancy.
- Explore the forecasts and classification results generated by the machine learning models.
